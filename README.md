[sample](https://kazuhi-ra.hatenablog.com/entry/2021/08/22/050850)

### How to use

```html
<head>
  <script src="https://unpkg.com/@kazuhi-ra/avw/dist/browser/index.js"></script>
</head>
<body>
  <ashitano-vest></ashitano-vest>
</body>
```

or

`npm i @kazuhi-ra/avw`

```jsx
import '@kazuhi-ra/avw'

const AshitanoVest = () => {
  return <ashitano-vest />
}
```
